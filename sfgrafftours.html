SF Graff Tours

A personal project that utilized the Instagram API and Google Maps API to display graffiti and street art markers throughout the city of San Francisco. Geolocation data and hashtag data was scraped from Instagram uploads and then displayed to markers that would populate a map of San Francisco. Users could then plot out walking tours of the city with the aim of seeing the hidden art gems that they never knew existed prior. SF Graff Tours was built in Sinatra as a mobile friendly one-page web application that heavily utilized AJAX to display a plethora of data to users in a clear and concise way.

Graff Tours SF was built in Sinatra as a one page application, utilizing Ruby, Javascript, Jquery, AJAX, HTML5 and CSS3. The Instagram API was used for authentication and retrieving user's photos. Google Maps API was used for all map, marker and geolocation data.
